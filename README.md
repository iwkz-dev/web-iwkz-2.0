This is a [Next.js](https://nextjs.org) app. We use pnpm for all commands (install, build, run).

## Prerequisites

- Node.js 22.x
- pnpm 10.x

## Install

```bash
pnpm install
```

## Develop

```bash
pnpm dev
```

Open http://localhost:3000 to view the app.

## Build

```bash
pnpm build
```

## Run production build

```bash
pnpm start
```

## Lint & Format

```bash
pnpm lint
pnpm format
```
